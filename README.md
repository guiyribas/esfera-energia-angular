# EsferaEnergiaAngular

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 10.0.5.

## Requisitos

Possuir o node.js instalado. Visite: https://nodejs.org/en/

## Como inicializar o projeto localmente passo a passo:

1. Acessar https://github.com/guiyribas/esfera-energia-angular e clonar todo o projeto.
2. Abrir o terminal e navegar até a pasta em que o projeto se encontra (ex.:
C:\Users\guiri\Documents\projetos\esfera-energia-angular)
3. Executar: npm install
4. Após instalado todas as dependências, dar start no projeto Angular. Executar: npm start
(equivalente ao comando ng serve)
