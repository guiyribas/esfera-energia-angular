# EsferaEnergiaAngular

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 10.0.5.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Requisitos

Possuir o node.js instalado. Visite: https://nodejs.org/en/

## Como inicializar o projeto localmente passo a passo:

1. Abrir o terminal e navegar até a pasta em que o projeto se encontra (ex.:
C:\Users\guiri\Documents\projetos\esfera-energia-angular)
2. Executar: npm install
3. Após instalado todas as dependências, dar start no projeto Angular. Executar: npm start
(equivalente ao comando ng serve)
